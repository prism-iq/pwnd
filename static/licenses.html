Data Sources

This application does not include any email data. Users must provide their own email corpus in compliance with applicable laws and regulations.

Important: Users are responsible for ensuring they have legal rights to analyze any data imported into this system.

Privacy & Data Processing

Local Processing: Query intent parsing is performed locally using Mistral 7B. No user queries are sent to external services during this step.

API Processing: Final analysis uses Claude Haiku API (Anthropic). Search results (not raw emails) are sent to Anthropic for synthesis. See Anthropic Privacy Policy.

No Telemetry: This application does not collect analytics, usage statistics, or telemetry data.
